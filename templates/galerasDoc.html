API Galera

Obtener Información de una Galera

URL: /gObtain

Método: POST

Parámetros:

  • numGalera (requerido): Número de la galera.
  • numLote (requerido): Número del lote.

Respuesta: Devuelve la información de la galera especificada en formato JSON.

Ejemplo de solicitud:

POST /gObtain
{
  "numGalera": '1',
  "numLote": 4
}

Ejemplo de respuesta:

{
  "error": 202,
  "message": "Good Job",
  "data": [
    {
      "idGalera": 1,
      "existence": 100,
      "typeChicken": "Pollo A"
"edad": "16"     }   ]
}

Obtener Galeras con base en su lote

URL: /galeras

Método: POST

Parámetros:

  • numLote (requerido): Número del lote.

Respuesta: Devuelve las galeras relacionadas con el número de lote especificado en formato JSON.

Ejemplo de solicitud:

POST /galeras
{
  "numLote": 456
}

Ejemplo de respuesta:

{
  "error": 202,
  "message": "Good Job",
  "data": [
    {
      "idGalera": 1,
      "existence": 100,
      "typeChicken": "Pollo A"
    },
    {
      "idGalera": 2,
      "existence": 200,
      "typeChicken": "Pollo B"
"numeroGalera": "2", "ca": 3.2, "idLote": 456     }
  ]
}

Asignación de galeras

URL: /asignate

Método: POST

Parámetros:

  • numLote (requerido): Número del lote.
  • numGalera (requerido): Número de la galera.
  • idTrabajador (requerido): ID del trabajador.
  • fecha_inicio (requerido): Fecha de inicio de operaciones.

Respuesta: Devuelve la asignación realizada.

Ejemplo de solicitud:

POST /asignate
{
  "numLote": 456,
  "numGalera": 2356,
  "idTrabajador": "001",
  "fecha_inicio": "23/10/2023"
}

Ejemplo de respuesta:

{
  "error": 202,
  "message": "Good Job",
}

Ejemplo de respuesta de error:

{
  "message": "Error al colocar los datos debias mandar como parametros para una nueva asignacion"
}

Crear neuva galera

URL: /gcreate

Método: POST

Descripción: Opción para poder ingresar una nueva galera a la base de datos.

Repuesta: Devuelve la asignación realizada.

Parámetros:

  • idGalera (requerido): ID de la nueva galera.
  • idLote (requerido): ID del lote asignado.
  • numeroGalera (requerido): Número de la nueva galera.
  • existencia (requerido): Cantidad de pollos dentro de la nueva galera.
  • tipo_pollo (requerido): Pollo macho, hembra o mixto.
  • idTrabajador (requerido):ID del trabajador asignado a la galera.
  • fecha_inicio (requerido): Fecha de inicio de actividades.

Respuesta: Devuelve la asignación realizada.

Ejemplo de solicitud:

POST /gcreate
{
  "idGalera": "2103",
  "idLote": "486",
  "numeroGalera": 3,
  "existencia": 300
  "tipo_pollo": "Mixto",
  "idTrabajador": "002",
  "fecha_inicio": "23/10/2023",
}

Ejemplo de respuesta:

{
  "error": 202,
  "message": "Good Job",
}

Ejemplo de respuesta de error:

{
  "message": "Error al colocar los datos debias mandar como parametros para la creacion de la nueva galera"
}
