API Galería

Obtener Información de una Galería

URL: /gObtain

Método: POST

Parámetros:

  • numGalera (requerido): Número de la galera.
  • numLote (requerido): Número del lote.

Respuesta: Devuelve la información de la galera especificada en formato JSON.

Ejemplo de solicitud:

POST /gObtain
{
  "numGalera": '1',
  "numLote": 4
}

Ejemplo de respuesta:

{
  "error": 202,
  "message": "Good Job",
  "data": [
    {
      "idGalera": 1,
      "existence": 100,
      "typeChicken": "Pollo A"
"edad": "16"     }   ]
}

Obtener Galerías

URL: /galeras

Método: POST

Parámetros:

  • numLote (requerido): Número del lote.

Respuesta: Devuelve las galerías relacionadas con el número de lote especificado en formato JSON.

Ejemplo de solicitud:

POST /galeras
{
  "numLote": 456
}

Ejemplo de respuesta:

{
  "error": 202,
  "message": "Good Job",
  "data": [
    {
      "idGalera": 1,
      "existence": 100,
      "typeChicken": "Pollo A"
    },
    {
      "idGalera": 2,
      "existence": 200,
      "typeChicken": "Pollo B"
"numeroGalera": "2", "ca": 3.2, "idLote": 456     }
  ]
}
