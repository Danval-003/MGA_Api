API Documentation

Route: /makeRegister

Method: POST

Description: Registers a new element in the database.

Parameters:

  • cantidadAlimento (required): Amount of food.
  • decesos (required): Number of deaths.
  • observaciones (required): Observations of the register.
  • idGalera (required): ID of the galera.
  • pesado (required): Boolean value indicating if the register has been weighed.

Response: Returns a JSON message indicating the result of the operation.

Example Request:

POST /makeRegister
{
  "cantidadAlimento": 500,
  "decesos": 2,
  "observaciones": "Some observations",
  "idGalera": 123,
  "pesado": true
}

Example Response:

{
  "error": 202,
  "message": "Good Job",
  "data": []
}
