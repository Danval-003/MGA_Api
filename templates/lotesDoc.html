API Lotes

Obtener Lotes

URL: /lObtain

Método: POST

Parámetros:

  • dpiTrabajador (requerido): DPI del trabajador.

Respuesta: Devuelve los lotes relacionados con el DPI del trabajador especificado en formato JSON.

Ejemplo de solicitud:

POST /lObtain
{
  "dpiTrabajador": "1234567890123"
}

Ejemplo de respuesta:

{
  "error": 202,
  "message": "Good Job",
  "data": [
    {
      "idLote": 1
    },
    {
      "idLote": 2
    }
  ]
}
